# 김비서 — 업무 대시보드

마케팅팀의 업무 효율성을 높이기 위한 올인원 대시보드 솔루션입니다.

---

## 🎯 주요 기능

### 📋 할 일 관리
- 우선순위별 할 일 추가/삭제 (🔴높음 🟡중간 🟢낮음)
- 완료 상태 체크박스 (줄이 그어지는 애니메이션)
- 담당자별 업무 현황 추적
- 필터링 (전체/높음/중간/낮음/완료)

### 📅 일정 관리
- 주간 캘린더 뷰 (요일별 이벤트)
- 월간 캘린더 (4월)
- 마감일 경고 표시 (⚠️)
- 우선순위별 색상 코딩

### 📊 매출 현황
- 제품별 매출 막대 차트
- 지역별 매출 현황
- 날짜별 추이 라인 차트
- 카테고리별 비중 도넛 차트
- CSV 데이터 기반 분석

### 📁 프로젝트 추적
- 5개 프로젝트 진행률 시각화
- 프로젝트별 상세 현황 (담당자/마감일)
- 상태 배지 (진행중/주의/초기단계)

### 👥 팀 현황
- 팀원별 업무 현황 카드
- 개인별 할 일 목록
- 주간 MVP 표시

### ⚙️ 설정
- 사용자 정보 관리
- 알림 토글
- 화면 설정 (시작 화면, 날짜 형식)

## 📁 파일 구조

```
2-체험용-샘플/
├── 업무대시보드.html              # 메인 대시보드 (SPA 구조)
├── 매출데이터.csv                 # 매출 원본 데이터
├── 매출데이터_시각화.html         # 매출 분석 차트 페이지
├── 마케팅_업무흐름.svg            # 마케팅 업무 플로우 다이어그램
├── subtrac_분석보고서.html        # SubTrackr 사이트 분석 보고서
├── 회의록.txt                      # 3월 9일 마케팅팀 회의록
└── 행사포스터.jpg                 # 샘플 이미지
```

### 파일별 역할

| 파일명 | 용도 |
|--------|------|
| **업무대시보드.html** | 메인 웹 애플리케이션 (7개 페이지 포함) |
| **매출데이터.csv** | 2026년 1~2월 판매 데이터 (30건) |
| **마케팅_업무흐름.svg** | 기획→제작→검토→배포→분석 프로세스 다이어그램 |
| **회의록.txt** | 업무 현황 및 할 일 기초 데이터 |

## 🛠 사용 기술

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 다크모드 + 반응형 그리드 레이아웃
- **JavaScript (Vanilla)**:
  - 상태 관리 (할 일, 프로젝트)
  - 라우팅 (SPA 페이지 전환)
  - 데이터 필터링 및 정렬

### 라이브러리
- **Chart.js**: 차트 시각화
  - 막대 차트 (제품별/지역별 매출)
  - 라인 차트 (추이)
  - 도넛 차트 (비중)

### 디자인 시스템
- **색상 변수** (CSS Custom Properties)
  - `--accent`: #58a6ff (파랑)
  - `--green`: #3fb950 (초록)
  - `--red`: #f85149 (빨강)
  - `--purple`: #bc8cff (보라)
  - `--yellow`: #d29922 (노랑)

- **다크모드**: GitHub 다크테마 기반
- **반응형**: 900px 이하 모바일 최적화

### 데이터 흐름
```
CSV (매출데이터) → JavaScript 배열 → Chart.js 렌더링
회의록.txt → 할 일 초기 데이터 → 동적 UI 업데이트
```

## 📱 페이지 구성 (7개)

1. **대시보드** - KPI + 할 일 + 일정 + 매출 한눈에 보기
2. **할 일 관리** - 우선순위별 필터링 & 통계
3. **매출 분석** - 차트 기반 상세 분석
4. **프로젝트** - 5개 프로젝트 진행률 추적
5. **캘린더** - 월간 달력 & 주요 일정
6. **팀 현황** - 팀원별 업무 현황
7. **설정** - 사용자/알림/화면 설정
